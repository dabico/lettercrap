API:

You can try the API in the console for yourself. Here's some examples of how to interact with it: 

            
const divs = document.querySelectorAll('div');
const [ image, text ] = divs;
Lettercrap.resetElement(image); // reset a single element
Lettercrap.initElement(image);  // initialize a single element
Lettercrap.resetElements(divs); // reset multiple elements
Lettercrap.initElements(divs);  // initialize multiple elements
Lettercrap.reset();             // reset initialized elements
Lettercrap.init();              // initialize elements
